这是一款学生考试成绩统计系统
登陆系统:
密码默认为"admin"，输入后可使用程序
它的功能分别有：
1. 添加学生:
	将想要添加的学生成绩以及姓名分别输入(例:小明 100)，后续可以在别的功能上查询或修改（成绩范围为0分~100分）
2. 查看学生:
	使用此功能可以查询输入的所有学生以及成绩
3. 修改学生:
	输入想要修改的学生以及修改后的成绩即可将将此学生的成绩进行修改
4. 删除学生:
	输入想要删除的学生姓名后，改学生的信息就会被删去
5. 保存文件:
	将现有信息保存起来，当下次登录时会自动加载保存的学生信息
6. 查询特定学生:
	会有三个选项分别为:
		1.查询成绩为(输入成绩数)的学生:
				会统计成绩为输入成绩数的学生姓名
		2.查询成绩为(输入范围)的学生(例:96 100):
				会统计成绩范围内的学生名字以及成绩(输入范围时不要在中间加标点符号）
		3.通过姓名或者相关汉字查询(例:明 姓名:小明 成绩:100):
				会通过输入的汉字查找是否有相关的姓名
7. 清空:
	清空保存的信息，再次登陆时不会自动加载
8. 成绩统计:
	统计方式共有3种:
		1.平均分:
			会结合所以学生成绩计算出平均分
		2.最高分:
			通过统计查询出所有学生分数最高的学生及成绩
		2.最低分:
			通过统计查询出所有学生分数最低的学生及成绩
9. 批量添加学生
	首先输入想要添加的学生数，后再去写添加学生的姓名与成绩
10.学生排序
	排序方式共4种分别为:
		1.成绩统计从高到低
				会将保存的学生成绩按高到低排序
		2.成绩统计从低到高
				会将保存的学生成绩按低到高排序
		3.姓名统计从高到低
				会按照学生姓名汉字按高到低顺序排序
		4.姓名统计从低到高
				会按照学生姓名汉字按低到高顺序排序
0. 退出
	推出该程序


使用流程:
	1,进入程序输入密码(默认admin)
	2,当弹出选项时输入数字选项来进行成绩(输入数字)
	3,当统计完时可以选择"保存"选项来保存你的学生信息,再次开启时会自动加载
	4,当想要输入新的学生信息时可以选择"清除",这样会将学生信息全部删除,再次开启时将不会加载
	5,做完所有的保存或者添加信息后,如果不许再继续使用,可以输入"0"来退出程序
